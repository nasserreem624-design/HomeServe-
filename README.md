
# HomeServe Pro Landing Page

هذا المشروع عبارة عن صفحة هبوط احترافية لخدمات الصيانة المنزلية في مصر، مبنية باستخدام React و Tailwind CSS و Framer Motion.

## خطوات التشغيل محلياً
1. قم بتحميل الأكواد في مجلد جديد.
2. تأكد من وجود Node.js مثبت على جهازك.
3. افتح الـ Terminal في المجلد واكتب:
   ```bash
   npm install
   npm run dev
   ```

## خطوات الرفع على GitHub و Vercel
1. **GitHub:** 
   - افتح حسابك واعمل New Repository باسم `homeserve-pro`.
   - ارفع الملفات باستخدام الأوامر التالية:
     ```bash
     git init
     git add .
     git commit -m "First production-ready version"
     git branch -M main
     git remote add origin [رابط الـ repo بتاعك]
     git push -u origin main
     ```
2. **Vercel:**
   - سجل دخولك في Vercel وادخل على Dashboard.
   - اضغط "Add New" ثم "Project".
   - اربط حساب GitHub واختار `homeserve-pro`.
   - اضغط Deploy وانتظر دقيقة.

مبروك! موقعك الآن متاح للجميع أونلاين.
